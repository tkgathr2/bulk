## 変更内容（必須）

### 1) 文言（全画面共通）

- ヘッダーやタイトル等の表示名を **「一括検索君」** に統一
  - 例：`Bulk Search` / `BulkSearch` / `Bulk Search —` などを置換
- フッターの表記も同様に統一

### 2) デザイン（Google寄せ）

次の方向性で `mockups/styles.css` を調整する（厳密な再現は不要。雰囲気を寄せる）。

- 背景：薄いグレー（例：`#f8f9fa` 系）
- カード：白背景＋薄い境界線＋控えめシャドウ
- フォント：Windows では `Segoe UI` のままでも可だが、優先順に `Roboto` を入れる（ローカル依存でOK、CDN禁止）
  - 例：`font-family: Roboto, "Segoe UI", "Meiryo", "Hiragino Sans", sans-serif;`
- 角丸：8px 前後、過度に丸くしない
- アクセント：Google青（例：`#1a73e8` 系）
- ボタン：青の塗りボタン（primary）＋白背景のアウトライン（secondary）
- タブ：下線アクティブ、ホバーは控えめ
- 入力欄：白背景、薄い枠線、フォーカス時に青枠

### 3) `results.html` のタブ

現状の「すべて」タブは残してよいが、Google寄せの見た目（下線＋バッジ）に調整する。
